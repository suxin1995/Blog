## airflow依赖关系

1. **方式一：set_downstream**

```python
# 方式一：
# 设置下游
t1.set_downstream(t2)

# 这意味着t2会依赖于t1运行成功后开始运行 
# 下面的设置是等效的
t2.set_upstream(t1)
```



2. **方式二：位运算符**

```python
'''
位运算符概念：位运算符用来对二进制位进行操作，也可用于链式运算
在Python中，位运算符包括位
与（&）、位或（|）、位求反（~）、位异或（^）、左移位（<<）和右移位（>>）
'''
# >>左侧先执行 右侧依赖左侧    <<小于号相反
t1 >> t2
t2 << t1
t1 >> t2 >> t3
```



3. **方式三:任务列表**

```python
# 任务列表也可以设置为依赖项。这些操作都有相同的效果
# [t2,t3]  t2,t3为一组，执行顺序随机
t1.set_downstream([t2, t3])
t1 >> [t2, t3]
[t2, t3] << t1
```
